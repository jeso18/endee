# 🎬 Movie Recommendation System

A Machine Learning project that recommends movies to users based on their preferences using **Collaborative Filtering** and **Cosine Similarity**.

---

## 📌 Overview

This project builds a movie recommender system using user rating data.  
It suggests movies by finding users with similar tastes and recommending what they liked.

---

## 🧠 How It Works

1. Load movie and rating data  
2. Create a **user-movie matrix**  
3. Compute similarity between users using **cosine similarity**  
4. Recommend movies liked by similar users  

---

## 📊 Dataset

- MovieLens Dataset  
- Contains:
  - `movies.csv` → movie titles & genres  
  - `ratings.csv` → user ratings
